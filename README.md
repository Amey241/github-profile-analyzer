# 🔭 GitHub Profile Analyzer

An institutional-grade developer insights platform that transforms raw commit history into deep personality and quality metrics. Built with **Streamlit**, **PyGithub**, and **Plotly**.

### 🔥 High-Impact Features
- 🎨 **Shareable Wrapped Card**: Generate high-fidelity summary images for social media.
- ⚔️ **Developer vs Developer Mode**: Overlay tech stacks and compare coding compatibility.
- 📝 **Commit Quality Scorer**: A rubric-based A–F grade for Git hygiene (imperative mood, length, references).
- 📦 **Repo Health Dashboard**: Real-time scorecard for documentation, CI/CD, license, and maintenance.

### 🛠️ Tech Stack
- **Dashboard**: Streamlit
- **NLP**: TextBlob (Sentiment), Sklearn (LDA Topic Modeling)
- **Visuals**: Plotly (Radar, Heatmap), WordCloud
- **Image Gen**: Pillow (Card Generator)
- **API**: PyGithub

### 🚀 Deployment
This project is configured for **Hugging Face Spaces** via the provided `Dockerfile` and `README.md` frontmatter.
